Problem 3: Fibonacci Number

Description:
Write a function that returns the nth Fibonacci number using iteration.
Assume the sequence starts as 0, 1, 1, 2, 3, 5, 8...

Function Signature:
def fibonacci(n: int) -> int:
Input:
- n: a positive integer representing the position (1-indexed)
Output:
- The nth Fibonacci number

Examples:
fibonacci(1) → 0
fibonacci(2) → 1
fibonacci(7) → 8

Test Cases:
assert fibonacci(1) == 0
assert fibonacci(2) == 1
assert fibonacci(7) == 8
assert fibonacci(10) == 34
